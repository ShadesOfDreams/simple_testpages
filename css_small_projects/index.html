Radial menu

  • init with RadialMenu(parameter)
  • parameter:
    • array
      • array = sub menu's list
      • array element has to be an object
      • array element parameters: title, class(optional)
      • default names will be used:
        • menu id = "radial-menu"
        • main menu class = "main-menu"
        • submenu container class = ".sub-menu-container"
    • object
      • bject is the options
      • object.list = submenu's list
      • object.menu = menu DOM element / selector
      • object.main = main menu DOM element / selector
      • object.subList = submnu container DOM element / selector
      • object.subItemClassName = submnu's DOM elements / selector
